# &#x09;				Operating System









#### **Main functions of an OS:-**





* ##### **Process Management :-**

###### &#x09;Creates, schedules \& terminates process 

###### &#x09;Handling multitasking \& CPU allocation.



* ##### **Memory Management :-**	

###### &#x09;Allocates \& deallocates RAM

###### &#x09;Manages Virtual Memory (Paging, Segmentation)

###### 

* ##### **File System Management :-**

###### &#x09;Organizes files and directories	

###### &#x09;Controls read/write permissions.

###### 

* ##### **Device Management :-**

###### &#x09;Manages Input/Output devices using drivers

###### &#x09;Uses buffering , caching and spooling.

###### 

* ##### **Security \& Protection :-**

###### &#x09;User authentication

###### &#x09;Access control \& permissions.

###### 

* ##### **User Interface :-**

###### &#x09;CLI(Command Line Interface ) e.g. Linux Terminal

###### &#x09;GUI(Graphical User Interface) e.g. Windows.

#### **Types of Operating Systems :-**

###### 

###### 

* ##### **Batch O/S :-**

###### &#x09;Jobs processed in batches.

###### &#x09;Ex: Early IBM systems.

###### 

* ##### **Time sharing O/S :-**

###### &#x09;Multiple users share CPU time.

###### &#x09;Ex: UNIX.

###### 

* ##### **Multiple Programming O/S :-**

###### &#x09;Multiple programs in memory at the same time.

###### &#x09;Ex: Linux, Windows.

###### 

* ##### **Multitasking O/S :-**

###### &#x09;Multiple tasks executed simultaneously.

###### &#x09;Ex: Windows, macOS.

###### 

* ##### **Real-Time OS (RIOS) :-**

###### &#x09;Strict time constraints.

###### &#x09;Ex: VX works, RT Linux.

###### 

* ##### **Distributed OS :-**

###### &#x09;Manages multiple systems as one.

###### &#x09;Ex: Amoeba.

* ##### **Client Operating System :-**

###### &#x09;		A client OS is an operating system used on personal computers or devices to allow users to run applications and access network services.

###### 

* ##### **Main Features :-**

###### 

###### &#x09;Graphical User Interface (GUI) for easy use

###### 

###### &#x09;Runs desktop applications

###### 

###### &#x09;Supports hardware devices (keyboard, mouse, printer)

###### 

###### &#x09;Can connect to networks and servers

###### 

###### &#x09;Usually used by one user at a time

###### 

###### 

###### **Examples :-** Windows 11, Ubuntu Desktop, macOS, Kali Linux (when used as a desktop)

###### 

###### 

###### **Common Uses:-** Web browsing, Document editing, Media Playback, Programming and basic development.

###### 

###### 

###### 

######

* ##### **Server Side Operating System :-**

###### &#x09;				A server-side operating system is an OS designed to manage network resources and provide services to multiple client computers. like files, 						databases,websites, and user access.

###### 

* ##### **Main Features :-**

###### 

###### &#x09;Supports multiple users at the same time

###### 

###### &#x09;Manages network resources

###### 

###### &#x09;Provides security and access control

###### 

###### &#x09;Runs server applications (web server, database server)

###### 

###### &#x09;High stability and performance

###### 

###### 

###### &#x09;**Examples :-** Windows Server, Ubuntu Server, Red Hat Enterprise Linux, CentOS.

###### 

###### 

###### &#x09;**Common Uses :-** Hosting websites and applications, File and database servers, Email and cloud services, Network management \& authentication.
